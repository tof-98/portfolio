## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tof-98/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:3000` (or the port Astro assigns).

## Building for Production

To create an optimized production build:

```bash
npm run build
```

The output will be in the `dist/` directory.
